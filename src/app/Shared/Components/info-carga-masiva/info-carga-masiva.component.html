¿Cómo debe estar estructurado mi archivo de turnos?

Ten en cuenta la siguiente información para realizar la creación o actualización de turnos de manera correcta:

  1. El archivo DEBE estar en formato .txt
  2. Todos los campos deberán ir separados por comas.
  3. Evita dejar espacio entre las comas.
  4. Cada turno deberá ir en una línea distinta.
  5. Evita dejar espacios vacíos antes o después de cada línea.
  6. Evita dejar alguna línea vacía antes del primer registro.
  7. Evita dejar alguna línea vacía después del último registro.
  8. Si deseas crear un turno, el campo correspondiente al ID del turno,
    deberá ir con 'null' (Sin comillas).
  9. Si deseas editar un turno existente, es necesario que indiques
    el ID del turno que deseas modificar.
  10. Es obligatorio indicar el documento del contratista a quien vas a asignar el turno.
  11. Si el turno es de Campo, deberás ingresar la misma fecha tanto inicial como final.
  12. Si el turno es de Oficina, deberás ingresar un rango de fechas (Fecha inicial y Fecha final distintas).

La estructura correcta para la creación del archivo es:

ID TURNO,DOCUMENTO CONTRATISTA,FECHA INICIAL, FECHA FINAL,JORNADA,HORA INICIO,HORA FIN,LABOR
